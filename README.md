# 💅 Ateliê Tati Brandão — Sistema de Gestão Financeira

Sistema web desenvolvido para controle financeiro de um ateliê, permitindo o gerenciamento de ganhos, vales, despesas e fluxo de caixa de forma simples e intuitiva.

---

## 📌 Sobre o projeto

O sistema foi criado para atender a necessidade de controle financeiro de uma equipe de colaboradoras em um salão, separando acessos entre:

* 👩‍💼 **Colaboradoras**
* 👑 **Proprietária (Salão)**

A aplicação é totalmente responsiva e otimizada para uso em dispositivos móveis.

---

## 🚀 Funcionalidades

### 🔐 Login

* Seleção de usuário (colaboradora ou proprietária)
* Autenticação simples

---

### 👩‍💼 Área da Colaboradora

* 📊 Visualização de saldo do dia
* 💰 Saldo da quinzena
* 📈 Ganhos brutos
* ➕ Registro de:

  * Ganhos
  * Vales (adiantamentos)
  * 📜 Histórico por quinzena

---

### 👑 Área da Proprietária

* 📊 Painel financeiro geral
* 💰 Cálculo automático de 30% do salão
* 👥 Gestão da equipe
* 💸 Registro de despesas
* 🧾 Controle de caixa
* 📅 Fechamento de caixa diário
* ✏️ Edição de registros via modal
